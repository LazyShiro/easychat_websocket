easychat_websocket
===============

> 运行环境要求PHP7.1+。

## 说明

* websocket服务框架使用swoft 2.*

## 使用

* cp .env.example .env
* php bin/swoft entity:create -d -y
* php bin/swoft ws:start -d
* [简单聊天室](https://github.com/LazyShiro/easychat)项目的WS服务支持

## 注意

* 本项目公开但不商用，如有商用请更新本项目现有BUG...
* 不要随意composer update
* 禁止吐槽代码质量
